Get Started in Minutes

Choose your preferred installation method and start using AI Runner today.

üì¶ Flatpak Installation (Linux)

Step 1: Install Flatpak

If you don't have Flatpak installed, install it first:

# Debian/Ubuntu
sudo apt install flatpak

# Fedora
sudo dnf install flatpak

# Arch
sudo pacman -S flatpak

Step 2: Add Flathub Remote

flatpak remote-add --if-not-exists --user flathub https://dl.flathub.org/repo/flathub.flatpakrepo

Step 3: Build & Install AI Runner

While our Flathub submission is pending approval, you can build and install manually:

# Clone the repository
git clone https://github.com/Capsize-Games/airunner.git
cd airunner/flatpak

# Install flatpak-builder if needed
sudo apt install flatpak-builder # Debian/Ubuntu

# Build the Flatpak
./build.sh

# Install locally
./build.sh install

Step 4: Run AI Runner

flatpak run com.capsizegames.AIRunner
üìù Note: Once approved on Flathub, you'll be able to install directly with:
flatpak install flathub com.capsizegames.AIRunner

üêç pip Installation (Linux/Windows)

Prerequisites

Ensure you have Python 3.10+ and pip installed. CUDA toolkit recommended for GPU acceleration.

Step 1: Create Virtual Environment (Recommended)

python -m venv airunner-env

# Linux/macOS
source airunner-env/bin/activate

# Windows
airunner-env\Scripts\activate

Step 2: Install AI Runner

pip install airunner

Step 3: Download Models

airunner-setup

Step 4: Run AI Runner

airunner
üí° Tip: For GPU acceleration, ensure you have CUDA drivers installed and use the appropriate PyTorch version for your GPU.
